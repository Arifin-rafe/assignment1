## TypeScript Practice Problems

## Features
- Filter even numbers from an array
- Reverse a string
- Check data types using union types
- Access object properties with generics
- Update object data using interfaces
- Use classes and inheritance
- Find common values between two arrays

## Functions Included

1. Filter Even Numbers
Returns only even numbers from an array given by user.

2. Reverse String
Reverses a given string in ts.

3. Check Type
Check if a value is a string or number.

4. Get Object Property
object properties safely using generics.

5. Toggle Read Status
Adds an isRead property to a book object by default.

6. Person and Student Class
explains class inheritance in TypeScript.

7. Array Intersection
Find out the most common elements between two arrays.
